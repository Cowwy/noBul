Project Objective

Replication of Black Fly Beverage Company website. This only has one page and it does not have any animation or transitions added. It is created to solely exercise positioning of HTML element and CSS optimization.

The website uses combination of divison position and flexbox to achieve full width and fixed width. However, responsive web design had not been incorporated, therefore, this website remains static. The challenges in creating this website isn't the layout, but the usage of SVG graphics to present statistical data and creating a CSS design that is robust and reusable.

Mistakes

I have added too many CSS properties in one single class, making the class un-reusable in other parts of the website. Often have to recreate a class just for a simple adjustment or forced to use inline CSS.

There was no planning phase to make the website repsonive from the beginning and near the end of the page, the position didn't align as well as I wanted.
